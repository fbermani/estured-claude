# design-references — notas de uso

Estas carpetas son inspiración de diseño (exportadas de Stitch), no spec de producto — ver `CLAUDE.md` para el detalle de qué copy está prohibido reproducir de acá.

**Usar siempre `code.html`, no `screen.png`.** El `code.html` es la referencia completa y confiable en todas las carpetas. Varios `screen.png` (9 al 2026-07-10, casi todos en `stitch_estured_mvp_1ra parte/`) nunca se descargaron bien — su contenido real es el texto de error `<FIFE Image failed to fetch>` en vez de una imagen, y fueron borrados de este repo para no hacerle perder tiempo a una sesión futura intentando "leerlos" (ya pasó en el Ciclo 2, ver `GAPS.md`).

**Si una tarea necesita específicamente el `screen.png` de una carpeta que ya no lo tiene** (compará con `code.html`, que sigue estando siempre): decíselo al usuario explícitamente — que no hay imagen válida para esa referencia — y pedile una captura de pantalla o imagen en vez de asumir o inventar el diseño a partir de otra fuente.
